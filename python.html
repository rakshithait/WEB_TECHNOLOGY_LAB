Functions

In Python, a function is a group of related statements that performs a specific task. Functions help break our program into smaller and modular chunks. As our program grows larger and larger, functions make it more organized and manageable. Furthermore, it avoids repetition and makes the code reusable.

1.Keyword def that marks the start of the function header.

2.A function name to uniquely identify the function.

3.Function naming follows the same rules of writing identifiers in Python.

4.Parameters (arguments) through which we pass values to a function.

5.They are optional.

6.A colon (:) to mark the end of the function header.

7.Optional documentation string (docstring) to describe what the function does.

8.One or more valid python statements that make up the function body.

9.Statements must have the same indentation level (usually 4 spaces).

10.An optional return statement to return a value from the function.

Data Types

Data types are the classification or categorization of data items. It represents the kind of value that tells what operations can be performed on a particular data. Since everything is an object in Python programming, data types are actually classes and variables are instance (object) of these classes. Following are the standard or built-in data type of Python:

1.Numeric

2.Sequence Type

3.Boolean

4.Set

5.Dictionary
